# Akkusativ – The Accusative Case

> **Level:** A1 | **Netzwerk Neu A1 Reference:** Kursbuch Kapitel 2–4, Arbeitsbuch Kapitel 2–4 (p. 32, 52, 63, 91, 126)

---

## Overview

The **Accusative** is the **direct object case** — it marks the noun that is directly receiving the action of the verb.

> **Key question:** *Wen?* (Whom?) or *Was?* (What?) is being acted upon?

```
Ich kaufe einen Pullover.     → I am buying a sweater.        (was? → Akkusativ)
Er trinkt den Kaffee.         → He is drinking the coffee.    (was? → Akkusativ)
Sie hat einen Bruder.         → She has a brother.            (wen? → Akkusativ)
```

The **only article that changes** from Nominative to Accusative is the **Maskulinum** (der → den, ein → einen, kein → keinen). Feminine, neuter, and plural are **identical** to Nominative.

---

## Core Rules

### Definite Articles: Nominative vs. Accusative

| Gender | Nominative | Accusative | Change? |
|--------|-----------|------------|---------|
| Maskulinum | **der** | **den** | ✅ YES |
| Femininum | **die** | **die** | ❌ no |
| Neutrum | **das** | **das** | ❌ no |
| Plural | **die** | **die** | ❌ no |

---

### Indefinite Articles: Nominative vs. Accusative

| Gender | Nominative | Accusative | Change? |
|--------|-----------|------------|---------|
| Maskulinum | **ein** | **einen** | ✅ YES |
| Femininum | **eine** | **eine** | ❌ no |
| Neutrum | **ein** | **ein** | ❌ no |
| Plural | — | — | ❌ no |

---

### Negative Articles: Nominative vs. Accusative

| Gender | Nominative | Accusative | Change? |
|--------|-----------|------------|---------|
| Maskulinum | **kein** | **keinen** | ✅ YES |
| Femininum | **keine** | **keine** | ❌ no |
| Neutrum | **kein** | **kein** | ❌ no |
| Plural | **keine** | **keine** | ❌ no |

### Full Summary Table

| | **Nominativ** | **Akkusativ** |
|--|:---:|:---:|
| **der** (m) | der / ein / kein | **den / einen / keinen** |
| **die** (f) | die / eine / keine | die / eine / keine |
| **das** (n) | das / ein / kein | das / ein / kein |
| **die** (pl) | die / — / keine | die / — / keine |

> **Memory trick:** Only masculine articles get an **-n** added in Accusative.

## Personal Pronouns in Accusative

When a **pronoun** is the direct object, it also changes form:

| Nominative | Accusative | English |
|-----------|------------|---------|
| ich | **mich** | me |
| du | **dich** | you |
| er | **ihn** | him / it (m) |
| sie | **sie** | her / it (f) — unchanged |
| es | **es** | it (n) — unchanged |
| wir | **uns** | us |
| ihr | **euch** | you (pl.) |
| sie / Sie | **sie / Sie** | them / You — unchanged |

```
Er liebt mich.          → He loves me.
Ich kenne ihn.          → I know him.
Sie sieht uns.          → She sees us.
Kennst du ihn?          → Do you know him?
```

## Accusative with Possessive Pronouns

Possessive pronouns also follow the **ein/einen** pattern in Accusative:

| | **Nominativ (m)** | **Akkusativ (m)** | **Nominativ (f/n/pl)** | **Akkusativ (f/n/pl)** |
|--|--|--|--|--|
| ich | mein | **meinen** | meine / mein / meine | meine / mein / meine |
| du | dein | **deinen** | deine / dein / deine | deine / dein / deine |
| er/es | sein | **seinen** | seine / sein / seine | seine / sein / seine |
| sie | ihr | **ihren** | ihre / ihr / ihre | ihre / ihr / ihre |
| wir | unser | **unseren** | unsere / unser / unsere | unsere / unser / unsere |
| ihr | euer | **euren** | eure / euer / eure | eure / euer / eure |
| sie/Sie | ihr/Ihr | **ihren/Ihren** | ihre/Ihre / ihr/Ihr / ihre/Ihre | — |

> From Arbeitsbuch p. 91: *Wo ist **ihren** Bruder? / Suchen Sie **Ihre** Fernbedienung?* — possessives in Accusative.

```
Ich suche meinen Schlüssel.       → I'm looking for my key.     (der Schlüssel → meinen)
Hast du deine Tasche?             → Do you have your bag?       (die Tasche → deine, no change)
Er hat sein Handy.                → He has his phone.           (das Handy → sein, no change)
```

## Accusative with Prepositions (A1)

These prepositions **always** take Accusative:

| Preposition | Meaning | Example |
|-------------|---------|---------|
| **durch** | through | Wir gehen durch **den Park**. |
| **für** | for | Das ist für **dich**. |
| **gegen** | against | Er ist gegen **den Plan**. |
| **ohne** | without | Ich gehe ohne **meinen Bruder**. |
| **um** | around / at (time) | Um **den Bahnhof** / Um **drei Uhr**. |
| **bis** | until / up to | Bis **nächsten Montag**. |
| **entlang** | along | Gehen Sie **die Straße** entlang. |

> **Memory aid:** **DOFGUB** → Durch, Ohne, Für, Gegen, Um, Bis — always Accusative.

## Sentence Pattern: Nominative vs. Accusative in One Sentence

```
SUBJECT (Nominativ) + VERB + OBJECT (Akkusativ)

Der Mann        kauft       einen Pullover.
(Nom – subject)             (Akk – direct object)

Die Frau        trinkt      den Kaffee.
(Nom – subject)             (Akk – direct object)

Das Kind        isst        ein Brot.
(Nom – subject)             (Akk – direct object)
```

## Common Pitfalls for English Speakers

1. **Only masculine changes** — the most common mistake is changing *die* to *den* for feminine nouns. Don't.
2. **Predicate nominative stays Nominative** — after *sein*, the noun is NOT accusative:
   ```
   Er ist ein Arzt.      ✓  (Nominative after sein — predicate)
   Er hat einen Arzt.    ✓  (Accusative after haben — direct object)
   ```
3. **möchten vs. mögen:** Both take Accusative, but *möchten* = would like (polite request), *mögen* = to like (preference).
4. **Pronoun confusion — er vs. ihn:**
   ```
   Er kauft ihn.   → He (Nom) buys it (Akk — masculine noun).
   ```
5. **Preposition traps:** *für* always takes Accusative even when it feels like describing a state.
6. **Word order:** The accusative object usually comes after the verb, but with pronouns it can shift:
   ```
   Ich kaufe den Pullover.       → normal order
   Den Pullover kaufe ich.       → topicalised (emphasis) — still Akkusativ
   ```

## Practice Exercises

### Exercise 1 — Nominative or Accusative?

Identify the case of the underlined noun:

1. **Der Mann** kauft ein Brot.
2. Ich esse **den Salat**.
3. **Das Hotel** ist groß.
4. Sie kauft **einen Anzug**.
5. **Die Frau** trinkt **einen Kaffee**.
6. Ich suche **meine Tasche**.

<details>
<summary>Answers</summary>

1. **Nominativ** — subject (wer kauft?)
2. **Akkusativ** — direct object (was esse ich?)
3. **Nominativ** — subject (was ist groß?)
4. **Akkusativ** — direct object (was kauft sie?)
5. **Die Frau** = Nominativ (subject); **einen Kaffee** = Akkusativ (object)
6. **Akkusativ** — direct object (was suche ich?)

</details>

### Exercise 2 — Fill in the correct article

1. Ich kaufe _____ Pullover. (der Pullover)
2. Er trinkt _____ Kaffee. (der Kaffee)
3. Sie isst _____ Schokolade. (die Schokolade)

<details>
<summary>Answers</summary>

</details>

### Exercise 3 — Replace with accusative pronoun

1. Ich kenne den Mann. → Ich kenne _____.
2. Er liebt die Frau. → Er liebt _____.
3. Siehst du das Kind? → Siehst du _____?
4. Sie sucht mich. → (already pronoun) → *stays* _____.

<details>
<summary>Answers</summary>

</details>
