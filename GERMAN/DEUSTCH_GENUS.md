# Genus – Grammatical Gender in German

> **Level:** A1 | **Netzwerk Neu A1 Reference:** Kursbuch Kapitel 1–3 / Arbeitsbuch Kapitel 1–3 (p. 31, 52, 110, 126, 139)

---

## Overview

Every German noun has a **grammatical gender** — Maskulinum, Femininum, or Neutrum. Unlike English, gender is not always predictable from meaning. The gender is shown by the **definite article** (der / die / das) and **must be memorised** with every noun.

> **Golden Rule:** Always learn a noun together with its article. Not just *Haus* but *das Haus*.

---

## Core Rules

### The Three Genders

| Gender               | Definite Article | Indefinite Article | Example                      |
| -------------------- | ---------------- | ------------------ | ---------------------------- |
| Maskulinum (m)       | **der**          | **ein**            | der Mann (the man)           |
| Femininum (f)        | **die**          | **eine**           | die Frau (the woman)         |
| Neutrum (n)          | **das**          | **ein**            | das Kind (the child)         |
| Plural (all genders) | **die**          | **—**              | die Männer / Frauen / Kinder |

> Note: Plural always uses **die**, regardless of gender.

---

## Gender Clues by Word Ending

While exceptions exist, these endings are reliable ~80–90% of the time:

### Maskulinum (der) — Common Endings

| Ending                 | Example                              | English                  |
| ---------------------- | ------------------------------------ | ------------------------ |
| **-er** (agent/tool)   | der Lehrer, der Computer             | teacher, computer        |
| **-or**                | der Motor, der Direktor              | engine, director         |
| **-ismus**             | der Tourismus, der Realismus         | tourism, realism         |
| **-ist**               | der Tourist, der Journalist          | tourist, journalist      |
| **-ant / -ent**        | der Praktikant, der Student          | intern, student          |
| **-ig / -ich / -ling** | der König, der Teppich, der Lehrling | king, carpet, apprentice |

### Femininum (die) — Common Endings

| Ending                 | Example                             | English                       |
| ---------------------- | ----------------------------------- | ----------------------------- |
| **-ung**               | die Wohnung, die Zeitung            | flat/apartment, newspaper     |
| **-heit**              | die Freiheit, die Gesundheit        | freedom, health               |
| **-keit**              | die Möglichkeit, die Freundlichkeit | possibility, friendliness     |
| **-schaft**            | die Mannschaft, die Freundschaft    | team, friendship              |
| **-ion**               | die Station, die Kommunikation      | station, communication        |
| **-tät**               | die Universität, die Qualität       | university, quality           |
| **-ik**                | die Musik, die Politik              | music, politics               |
| **-ie**                | die Familie, die Energie            | family, energy                |
| **-in** (female agent) | die Lehrerin, die Ärztin            | female teacher, female doctor |

### Neutrum (das) — Common Endings

| Ending                        | Example                      | English                   |
| ----------------------------- | ---------------------------- | ------------------------- |
| **-chen** (diminutive)        | das Mädchen, das Häuschen    | girl, little house        |
| **-lein** (diminutive)        | das Büchlein, das Kindlein   | little book, little child |
| **-ment**                     | das Dokument, das Instrument | document, instrument      |
| **-tum**                      | das Datum, das Wachstum      | date, growth              |
| **-um**                       | das Museum, das Stadium      | museum, stadium           |
| **Ge- prefix**                | das Gebäude, das Gespräch    | building, conversation    |
| **Infinitives used as nouns** | das Essen, das Lernen        | food/eating, learning     |

## Gender by Meaning Category

Some categories are almost always the same gender:

| Category              | Gender  | Examples                            |
| --------------------- | ------- | ----------------------------------- |
| Male persons          | **der** | der Vater, der Bruder, der Mann     |
| Female persons        | **die** | die Mutter, die Schwester, die Frau |
| Days, months, seasons | **der** | der Montag, der Januar, der Winter  |
| Compass directions    | **der** | der Norden, der Süden               |
| Car brands            | **der** | der BMW, der Mercedes               |
| Numbers, letters      | **die** | die Eins, die A                     |
| Colors as nouns       | **das** | das Rot, das Blau                   |
| Young/juvenile beings | **das** | das Kind, das Baby, das Mädchen\*   |

> \*das Mädchen = "girl" is **neuter** because of the **-chen** diminutive ending — the grammatical rule overrides biological gender.

## Sentence Examples

```
Das ist ein Bahnhof.          → That is a train station.       (der → ein)
Das ist eine Kirche.          → That is a church.              (die → eine)
Das ist ein Hotel.            → That is a hotel.               (das → ein)

Ich kaufe einen Pullover.     → I'm buying a sweater.          (Akkusativ – see AKKUSATIVE)
Wo ist die Straße?            → Where is the street?
Der Lehrer kommt aus Berlin.  → The teacher comes from Berlin.
```

## Common Pitfalls for English Speakers

1. **No gender in English** — there's no shortcut. You must memorise der/die/das with every noun.
2. **das Mädchen is neuter**, not feminine — grammar beats biology for diminutives (-chen/-lein).
3. **Plural is always die** — don't confuse plural *die* with feminine *die*.
4. **Articles change in different cases** (Nominative, Akkusativ, Dativ) — gender is fixed, but the article form changes. See [[DEUSTCH_AKKUSATIV]] and [[DEUSTCH_NOMINATIVE]].
5. **Compound nouns** take the gender of the **last noun**: der Bahnhof + die Straße → **die** Bahnhofstraße.
6. **Borrowed words** are tricky: das Hotel, das Restaurant (neuter), der Computer (masculine) — no firm rule.

### Exercise 1 — Fill in der / die / das

1. _____ Bahnhof ist groß.
2. _____ Kirche ist alt.
3. _____ Hotel ist neu.
4. _____ Lehrer kommt aus München.
5. _____ Jacke ist blau.
6. _____ Hemd ist weiß.
7. _____ Mädchen heißt Anna.
8. _____ Universität ist groß.
9. _____ Dokument ist wichtig.
10. _____ Freundschaft ist schön.

<details>
<summary>Answers</summary>

1. **der** Bahnhof (m)
2. **die** Kirche (f)
3. **das** Hotel (n)
4. **der** Lehrer (m)
5. **die** Jacke (f)
6. **das** Hemd (n)
7. **das** Mädchen (n — -chen rule)
8. **die** Universität (f — -tät ending)
9. **das** Dokument (n — -ment ending)
10. **die** Freundschaft (f — -schaft ending)

</details>
