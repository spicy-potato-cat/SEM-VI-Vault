# Nationalität & Sprache – Nationality & Language

> **Level:** A1 | **Netzwerk Neu A1 Reference:** Kursbuch Kapitel 1–2, Arbeitsbuch Kapitel 1–2 (p. 14)

---

## Overview

In German, nationalities and languages follow strict patterns:
- **Nationalities** have masculine and feminine forms — **no article** when used predicatively (*Ich bin Österreicher.*)
- **Languages** are usually derived from the feminine nationality form and used **without article** (*Ich spreche Deutsch.*)
- **Country names** are mostly used **without an article** — but a few always take one (*die Schweiz, die Türkei, die USA*)

## 2. Endings Pattern — How Nationality Words are Formed

Most nationalities follow predictable patterns:

### Masculine forms

| Ending | Country type | Example |
|--------|-------------|---------|
| **-er** | most countries | Österreich → Österreich**er**, Japan → Japan**er** |
| **-e** | Slavic/Romance countries | Polen → Pol**e**, Frankreich → Franz**ose** |
| *no ending* | adj-type (Deutsche) | Deutschland → Deutsch**e** (der Deutsche) |

### Feminine forms

All add **-in** to the masculine (with occasional spelling change):

| Masculine | Feminine |
|-----------|----------|
| der Österreicher | die Österreich**erin** |
| der Spanier | die Spanier**in** |
| der Pole | die Pol**in** (drops -e) |
| der Franzose | die Französ**in** (Umlaut + drops -e) |
| der Russe | die Russ**in** |
| der Türke | die Türk**in** |

## 3. Grammar Rules

### Rule 1: No article for nationalities used predicatively

```
Ich bin Österreicher.        → I am Austrian. (m)
Ich bin Österreicherin.      → I am Austrian. (f)
Er ist Franzose.             → He is French.
Sie ist Japanerin.           → She is Japanese.
```

> Compare with professions — same rule: *Er ist Arzt.* (no article)

### Rule 2: No article for languages after sprechen/lernen

```
Ich spreche Deutsch.         → I speak German.
Ich lerne Französisch.       → I am learning French.
Sie spricht Englisch und Spanisch.
Er versteht kein Russisch.   → He doesn't understand Russian.
```

### Rule 3: Country names — mostly WITHOUT article

```
Ich komme aus Deutschland.
Ich lebe in Österreich.
Ich fahre nach Frankreich.
```

### Rule 4: These countries ALWAYS take an article

| Country | Article | Example |
|---------|---------|---------|
| **die Schweiz** | die (f) | Ich komme aus **der Schweiz**. |
| **die Türkei** | die (f) | Er fährt in **die Türkei**. |
| **die USA** | die (pl) | Sie wohnt in **den USA**. |
| **die Niederlande** | die (pl) | Er kommt aus **den Niederlanden**. |
| **der Iran** | der (m) | Sie fährt in **den Iran**. |
| **der Irak** | der (m) | — |
| **der Sudan** | der (m) | — |

## 4. Useful Phrases — Woher kommen Sie?

From Arbeitsbuch p. 14:

```
— Woher kommen Sie?
— Ich komme aus Deutschland. / Ich bin aus Österreich.

— Welche Sprachen sprechen Sie?
— Ich spreche Deutsch und Englisch.

— Sind Sie Österreicher?
— Ja, ich bin Österreicher. / Nein, ich bin Schweizer.

— Welche Nationalität haben Sie?
— Ich bin Schweizerin. Ich komme aus Zürich.

— Wo wohnen Sie jetzt?
— Ich wohne jetzt in München.
```

From Arbeitsbuch p. 14 (course context — student introductions):
```
Ich heiße Sofia. Ich komme aus Spanien. Ich spreche Spanisch und Englisch.
Ich heiße Ali. Ich komme aus der Türkei. Ich spreche Türkisch und Arabisch.
Ich heiße Yuki. Ich komme aus Japan. Ich spreche Japanisch.
```

## Common Pitfalls for English Speakers

1. **No article for nationalities in predicative use:**
   ```
   Er ist ein Österreicher.   ✗ (sounds literary/emphatic)
   Er ist Österreicher.       ✓
   ```

2. **Nationality ≠ Language name:** *Österreicher* speaks *Deutsch*, not *Österreichisch*:
   ```
   Er ist Österreicher und spricht Deutsch.   ✓
   Er spricht Österreichisch.                  ✗
   ```

3. **die Schweiz / die Türkei — don't drop the article:**
   ```
   Ich komme aus Schweiz.       ✗
   Ich komme aus der Schweiz.   ✓
   ```

4. **nach vs. in die for travel:**
   ```
   Ich fahre nach Deutschland.    ✓ (no article country)
   Ich fahre in die Schweiz.      ✓ (article country → in + Akk)
   Ich fahre in die Türkei.       ✓
   Ich fahre in die USA.          ✓
   ```

5. **Languages are capitalized in German** (unlike many European languages):
   ```
   Ich spreche deutsch.      ✗
   Ich spreche Deutsch.      ✓
   ```
