# Zahlen – Numbers

> **Level:** A1 | **Netzwerk Neu A1 Reference:** Kursbuch Kapitel 2–3, Arbeitsbuch Kapitel 2–3 (p. 23, 72)

---

## Overview

Numbers in German are essential for telling time, dates, prices, phone numbers, ages, and addresses. Cardinal numbers (one, two, three…) are learned first; ordinal numbers (first, second…) are used for dates (see [[DEUSTCH_WEEKEND_AND_MONTH]]).

## 1. Kardinalzahlen — Cardinal Numbers

### 0–20

| # | German | # | German |
|---|--------|---|--------|
| 0 | **null** | 11 | **elf** |
| 1 | **eins** | 12 | **zwölf** |
| 2 | **zwei** | 13 | **dreizehn** |
| 3 | **drei** | 14 | **vierzehn** |
| 4 | **vier** | 15 | **fünfzehn** |
| 5 | **fünf** | 16 | **sechzehn** |
| 6 | **sechs** | 17 | **siebzehn** |
| 7 | **sieben** | 18 | **achtzehn** |
| 8 | **acht** | 19 | **neunzehn** |
| 9 | **neun** | 20 | **zwanzig** |
| 10 | **zehn** | | |

> Note: 16 = *sechzehn* (sechs drops the -s), 17 = *siebzehn* (sieben drops -en)

### 10–100 (Tens)

From Arbeitsbuch p. 23:

| # | German |
|---|--------|
| 10 | **zehn** |
| 20 | **zwanzig** |
| 30 | **dreißig** |
| 40 | **vierzig** |
| 50 | **fünfzig** |
| 60 | **sechzig** |
| 70 | **siebzig** |
| 80 | **achtzig** |
| 90 | **neunzig** |
| 100 | **hundert** |

> Note: 30 = *dreißig* (not dreißig — special spelling with -ßig not -zig), 60 = *sechzig* (sechs drops -s), 70 = *siebzig* (sieben drops -en)

### 21–99 (Combined Numbers)

Pattern: **ones + und + tens** (ones come FIRST, then *und*, then tens):

```
21 = einundzwanzig        (ein + und + zwanzig)
32 = zweiunddreißig       (zwei + und + dreißig)
43 = dreiundvierzig
54 = vierundfünfzig
65 = fünfundsechzig
76 = sechsundsiebzig
87 = siebenundachtzig
99 = neunundneunzig
```

From Arbeitsbuch p. 23 (practice numbers):
```
39 → neununddreißig
85 → fünfundachtzig
42 → zweiundvierzig
51 → einundfünfzig
63 → dreiundssechzig
76 → sechsundsiebzig
94 → vierundneunzig
```

> **English says "twenty-one" (tens first).**
> **German says "einundzwanzig" (ones first).**

## 2. Special Forms of Numbers

### eins vs. ein

| Context | Form | Example |
|---------|------|---------|
| Counting / standalone | **eins** | Eins, zwei, drei… |
| Before a noun | **ein/eine/ein** | ein Mann, eine Frau |
| Telephone | **eins** | null-acht-sieben-**eins** |

### zwei vs. zwo

In telephone numbers and when clarity is needed, **zwo** is used instead of *zwei* to avoid confusion with *drei*:
```
0878-zwo-null-drei → 0878-2-0-3
```

## 3. Rechnen — Basic Arithmetic

| Symbol | German | Spoken |
|--------|--------|--------|
| + | plus | drei **plus** vier ist sieben |
| − | minus | zehn **minus** drei ist sieben |
| × | mal | drei **mal** vier ist zwölf |
| ÷ | geteilt durch | zwölf **geteilt durch** vier ist drei |
| = | ist / gleich | drei plus vier **ist** sieben |

## 4. Preise — Prices

| German | English |
|--------|---------|
| Wie viel kostet das? | How much does it cost? |
| Das kostet … Euro. | It costs … euros. |
| … Euro fünfzig | … euros fifty |
| Stimmt so. | Keep the change. |
| Machen Sie … bitte. | That comes to … please. |

From Arbeitsbuch p. 76:
```
Machen Sie 12, bitte.     → That'll be 12, please.
Stimmt so.                → Keep the change.
```

Price format:
```
€12,50  → zwölf Euro fünfzig
€3,99   → drei Euro neunundneunzig
€100    → hundert Euro
€850    → achthundertfünfzig Euro
```

## 5. Telefonnummern — Phone Numbers

Phone numbers are read in **pairs** or **single digits**:

From Arbeitsbuch p. 10:
```
021xxxx    → null-zwei-eins-...
0878xxxx   → null-acht-sieben-acht-...
16454      → eins-sechs-vier-fünf-vier
             OR sechzehn-vier-vierundfünfzig
```

> Always say **null** for 0, never *zero*.

Quick reference for dates:

| 1. | erst- | 11. | elft- |
|---|------|---|------|
| 2. | zweit- | 12. | zwölft- |
| 3. | dritt- | 20. | zwanzigst- |

## Common Pitfalls for English Speakers

1. **Ones come BEFORE tens** in 21–99:
   ```
   21 = einundzwanzig   (not zwanzig-ein)
   45 = fünfundvierzig  (not vierzig-fünf)
   ```

2. **dreißig not dreißig** — 30 has a special spelling with **ß**, not *drizig* or *dreißig*.

3. **sechzehn / sechzig** — 6 drops the -s before *-zehn* and *-zig*.

4. **siebzehn / siebzig** — 7 drops the -en before *-zehn* and *-zig*.

5. **eins vs. ein** — standalone = *eins*; before a noun = *ein/eine*.

6. **Decimals use commas, not dots** in German:
   ```
   €12.50 (English) = €12,50 (German) = zwölf Euro fünfzig
   ```

7. **Thousands use dots or spaces**, not commas:
   ```
   1,000 (English) = 1.000 or 1 000 (German)
   ```
