# Beziehungen & Wegbeschreibung – Relationships & Directions

> **Level:** A1 | **Netzwerk Neu A1 Reference:** Kursbuch Kapitel 2–6, Arbeitsbuch Kapitel 2–6 (p. 31, 35, 89, 100–103)

---

## Overview

This topic covers two connected areas:
1. **Familie und Beziehungen** — family members and personal relationships
2. **Wegbeschreibung und Ortsangaben** — giving/understanding directions and describing locations

Both areas use **prepositions**, which in German require specific cases (Dativ for location, Akkusativ for direction).

```
Entschuldigung, wo ist der Bahnhof?
→ Excuse me, where is the train station?

Wie komme ich zum Rathaus?
→ How do I get to the town hall?

Wie weit ist es?
→ How far is it?

Ist es weit von hier?
→ Is it far from here?
```

## 4. Lokalpräpositionen — Prepositions of Place

### Static Location (Dativ) — Wo? Where is it?

These prepositions use **Dativ** when answering *Wo?* (location):

| Preposition | Meaning | Example |
|-------------|---------|---------|
| **in** + Dat | in / inside | im (in dem) Zimmer |
| **an** + Dat | at / on (vertical surface) | am (an dem) Fenster |
| **auf** + Dat | on (horizontal surface) | auf dem Tisch |
| **neben** + Dat | next to | neben dem Sofa |
| **zwischen** + Dat | between | zwischen der Küche und dem Bad |
| **vor** + Dat | in front of | vor dem Hotel |
| **hinter** + Dat | behind | hinter der Kirche |
| **über** + Dat | above / over | über dem Bett |
| **unter** + Dat | under / below | unter dem Tisch |

### Directed Movement (Akkusativ) — Wohin? Where to?

The same prepositions use **Akkusativ** when answering *Wohin?* (direction):

| Preposition | Meaning | Example |
|-------------|---------|---------|
| **in** + Akk | into | ins (in das) Kino |
| **an** + Akk | to / up to | an die Wand |
| **auf** + Akk | onto | auf den Tisch |
| **neben** + Akk | next to (placing) | neben das Sofa |
| **vor** + Akk | in front of (placing) | vor den Eingang |
| **hinter** + Akk | behind (placing) | hinter die Tür |
| **über** + Akk | above (going over) | über die Brücke |
| **unter** + Akk | under (placing) | unter den Tisch |

### Memory trick: Wo? = Dativ / Wohin? = Akkusativ

```
WO?   → DAT  (static: the thing is THERE)
WOHIN? → AKK  (motion: putting/going somewhere)

Das Buch liegt auf dem Tisch.      (Wo? — Dativ)
Ich lege das Buch auf den Tisch.   (Wohin? — Akkusativ)
```

## 6. Prepositions with Fixed Cases (not two-way)

Some prepositions **always take Dativ** (never two-way):

| Preposition | Always Dativ | Example |
|-------------|-------------|---------|
| **mit** | + Dat | mit dem Bus fahren |
| **bei** | + Dat | beim Arzt |
| **von** | + Dat | vom Bahnhof |
| **nach** | + Dat | nach Berlin fahren |
| **aus** | + Dat | aus Deutschland |
| **zu** | + Dat | zum Hotel, zur Schule |
| **seit** | + Dat | seit zwei Tagen |
| **gegenüber** | + Dat | gegenüber dem Bahnhof |

From Arbeitsbuch p. 89:
```
nach dem Meeting        → after the meeting (nach + Dat)
mit einer Kollegin      → with a colleague (mit + Dat)
von der Firma           → from the company (von + Dat)
zu den Chef             → to the boss (zu + Dat)
aus dem Marketing       → from marketing (aus + Dat)
beim Chef               → at the boss's (bei + Dat)
zum Essen               → to the meal (zu + Dat)
zur Chefin              → to the female boss (zu + Dat)
```

## Common Pitfalls for English Speakers

1. **Wo vs. Wohin determines the case:**
   ```
   Das Buch ist auf dem Tisch.    (Wo? → Dativ)
   Ich lege das Buch auf den Tisch.  (Wohin? → Akkusativ)
   ```

2. **im vs. ins:** *im* = in + dem (location); *ins* = in + das (direction into neuter noun):
   ```
   Ich bin im Kino.     (Wo? = location)
   Ich gehe ins Kino.   (Wohin? = direction)
   ```

3. **gegenüber** comes AFTER a pronoun but before/after a noun:
   ```
   gegenüber dem Bahnhof   ✓
   mir gegenüber            ✓ (after pronoun)
   ```

4. **nach vs. zu vs. in for destinations:**
   - *nach* + cities/countries (no article): nach Berlin, nach Deutschland
   - *zu* + people/institutions: zum Arzt, zur Schule
   - *in* + places with article: ins Kino, in die Stadt

5. **seit + present tense** for ongoing states (unlike English perfect):
   ```
   Ich lerne seit zwei Jahren Deutsch.
   → I have been learning German for two years. (present in German!)
   ```

You are at the Bahnhof. Give directions to the Rathaus:

**Map info:** Go straight for 200m, then turn left, then turn right at the Goethestraße, the Rathaus is opposite the Kirche.

### Exercise 3 — Wo? → Dativ or Wohin? → Akkusativ?

Fill in the correct article form:

1. Das Buch liegt auf _____ Tisch. (der Tisch — Wo?)
2. Ich stelle das Buch auf _____ Tisch. (der Tisch — Wohin?)
3. Die Lampe hängt über _____ Sofa. (das Sofa — Wo?)
4. Ich hänge die Lampe über _____ Sofa. (das Sofa — Wohin?)
5. Der Schlüssel liegt neben _____ Tür. (die Tür — Wo?)
6. Er legt den Schlüssel neben _____ Tür. (die Tür — Wohin?)

<details>
<summary>Answers</summary>

1. auf **dem** Tisch (Dativ — Wo?)
2. auf **den** Tisch (Akkusativ — Wohin?)
3. über **dem** Sofa (Dativ — Wo?)
4. über **das** Sofa (Akkusativ — Wohin?)
5. neben **der** Tür (Dativ — Wo?)
6. neben **die** Tür (Akkusativ — Wohin?)

</details>

### Exercise 4 — nach / zu / in for destinations

Fill in nach, zum, zur, ins, or in die:

1. Ich fahre _____ Berlin.
2. Ich gehe _____ Arzt.
3. Wir fahren _____ Österreich.
4. Er geht _____ Kino.
5. Sie fährt _____ Schule.
6. Ich gehe _____ Stadt.

<details>
<summary>Answers</summary>

1. **nach** Berlin (city, no article)
2. **zum** Arzt (zu + dem)
3. **nach** Österreich (country, no article)
4. **ins** Kino (in + das)
5. **zur** Schule (zu + der)
6. **in die** Stadt (in + die — with article)

</details>
