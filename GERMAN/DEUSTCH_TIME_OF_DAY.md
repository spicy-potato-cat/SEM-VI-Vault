# Uhrzeit & Tageszeit – Telling Time & Times of Day

> **Level:** A1 | **Netzwerk Neu A1 Reference:** Kursbuch Kapitel 3–4, Arbeitsbuch Kapitel 3–4 (p. 59–60)

---

## Overview

German has **two systems** for telling time:
- **Official / 24-hour** (used in travel, schedules, announcements)
- **Colloquial / 12-hour** (used in everyday conversation)

> **Key question for time:** *Wie spät ist es?* / *Wie viel Uhr ist es?* — What time is it?
> **Saying when something happens:** *Um wie viel Uhr...?* — At what time...?

---

## 1. Official Time (24-Stunden-System)

Simple: state the hour, then *Uhr*, then the minutes.

```
14:00  → vierzehn Uhr
9:30   → neun Uhr dreißig
18:45  → achtzehn Uhr fünfundvierzig
20:15  → zwanzig Uhr fünfzehn
```

| Time | German |
|------|--------|
| 8:00 | acht Uhr |
| 10:30 | zehn Uhr dreißig |
| 13:00 | dreizehn Uhr |
| 15:20 | fünfzehn Uhr zwanzig |
| 21:00 | einundzwanzig Uhr |
| 0:00 | null Uhr / Mitternacht |

From Arbeitsbuch p. 59–60:
```
14:00 → vierzehn Uhr (zwei mittags)
6:50  → sechs Uhr fünfzig (kurz nach sechs morgens)
10:07 → zehn Uhr sieben (kurz nach zehn vormittags)
11:30 → elf Uhr dreißig (halb zwölf vormittags)
9:14  → neun Uhr vierzehn (vierzehn nach neun)
3:45  → drei Uhr fünfundvierzig (viertel vor vier)
4:15  → vier Uhr fünfzehn (viertel nach vier)
4:10  → vier Uhr zehn (kurz nach vier)
14:09 → vierzehn Uhr neun (neun nach zwei mittags)
12:30 → zwölf Uhr dreißig (halb eins mittags)
```

## 2. Colloquial Time (12-Stunden-System)

Uses **relative expressions** around the clock:

### Key Phrases

| Expression | Meaning | Example |
|-----------|---------|---------|
| **... Uhr** | o'clock / exact hour | Es ist drei Uhr. |
| **Viertel nach ...** | quarter past | Viertel nach drei (3:15) |
| **halb ...** | half to (NOT half past!) | halb vier (3:30 — half to four) |
| **Viertel vor ...** | quarter to | Viertel vor vier (3:45) |
| **... nach ...** | ... minutes past ... | zehn nach drei (3:10) |
| **... vor ...** | ... minutes to ... | zehn vor vier (3:50) |
| **kurz nach ...** | just after | kurz nach drei (≈3:05) |
| **kurz vor ...** | just before | kurz vor vier (≈3:55) |
| **gegen ...** | around / about | gegen drei (≈3:00) |

### ⚠️ CRITICAL: halb = half TO, not half past

> English "half three" = 3:30 BUT German **halb drei** = **2:30** (halfway to three)

| Time | Colloquial German | Literal meaning |
|------|------------------|-----------------|
| 2:30 | **halb drei** | half (way) to three |
| 5:30 | **halb sechs** | half (way) to six |
| 12:30 | **halb eins** | half (way) to one |
| 8:30 | **halb neun** | half (way) to nine |

### Clock Face Summary

```
           12
      kurz vor 12    kurz nach 12
   Viertel vor 12      Viertel nach 12

  9                          3
   Viertel vor ...      Viertel nach ...

          halb ...
           6
```

| Clock position | Expression |
|----------------|-----------|
| :00 | ... Uhr |
| :05 | fünf nach ... |
| :10 | zehn nach ... |
| :15 | Viertel nach ... |
| :20 | zwanzig nach ... |
| :25 | fünf vor halb ... |
| :30 | halb ... (next hour!) |
| :35 | fünf nach halb ... |
| :40 | zwanzig vor ... |
| :45 | Viertel vor ... |
| :50 | zehn vor ... |
| :55 | fünf vor ... |

## 3. Tageszeit — Times of Day

| German | Meaning | Approximate hours |
|--------|---------|------------------|
| **der Morgen** | morning | 6:00–9:00 |
| **der Vormittag** | late morning / forenoon | 9:00–12:00 |
| **der Mittag** | midday/noon | 12:00–14:00 |
| **der Nachmittag** | afternoon | 14:00–18:00 |
| **der Abend** | evening | 18:00–22:00 |
| **die Nacht** | night | 22:00–6:00 |
| **die Mitternacht** | midnight | 0:00 |

### Adverbial Forms (time adverbs — add -s):

| Noun | Adverb | Meaning |
|------|--------|---------|
| der Morgen | **morgens** | in the morning(s) |
| der Vormittag | **vormittags** | in the late morning |
| der Mittag | **mittags** | at midday |
| der Nachmittag | **nachmittags** | in the afternoon |
| der Abend | **abends** | in the evening |
| die Nacht | **nachts** | at night |

From Arbeitsbuch p. 59:
```
kurz nach sechs morgens     → just after six in the morning
vierzehn nach neun vormittags  → fourteen past nine in the late morning
halb zwölf vormittags       → half eleven in the morning (= 11:30)
viertel vor vier nachts     → quarter to four at night (= 3:45)
```

## 4. Saying When — Time Phrases

| Expression | Meaning | Example |
|-----------|---------|---------|
| **um** + time | at (a specific time) | um acht Uhr, um halb drei |
| **von ... bis ...** | from ... to ... | von acht bis dreizehn Uhr |
| **ab** + time | from ... onwards | ab zehn Uhr |
| **bis** + time | until | bis 18 Uhr |
| **gegen** + time | around / about | gegen neun Uhr |
| **morgen** | tomorrow | morgen um drei |
| **heute** | today | heute Abend |
| **gestern** | yesterday | gestern Morgen |

From Arbeitsbuch p. 60 (Lea's schedule):
```
Am Montag und Freitag hat Lea Uni von acht bis dreizehn Uhr.
Am Dienstag von zehn Uhr dreißig bis achtzehn Uhr dreißig.
Am Donnerstag von acht bis achtzehn Uhr.
```

## Common Pitfalls for English Speakers

1. **halb ≠ half past** — the most dangerous mistake in German time:
   ```
   halb vier = 3:30   (NOT 4:30)
   halb elf  = 10:30  (NOT 11:30)
   ```

2. **um is required for "at" a specific time** — don't omit it:
   ```
   Der Kurs beginnt um neun Uhr.   ✓
   Der Kurs beginnt neun Uhr.      ✗
   ```

3. **Official time vs colloquial** — for schedules/transport always use 24h official; in conversation use colloquial.

4. **viertel nach vs. Viertel vor** — *nach* = past (moving away from hour), *vor* = to (approaching next hour).

5. **gegen ≠ um** — *gegen drei* means approximately 3:00; *um drei* means exactly 3:00.

### Exercise 4 — Netzwerk Neu A1 style: Lea's schedule

Answer the questions based on this schedule:

| Day | Time |
|-----|------|
| Montag | 8:00–13:00 |
| Dienstag | 10:30–18:30 |
| Donnerstag | 8:00–18:00 |
| Freitag | 8:00–13:00 |

1. Wann hat Lea am Montag Uni? → Von _____ bis _____.
2. Wie lange hat Lea am Dienstag Uni?
3. Hat Lea am Mittwoch Uni?
4. Um wie viel Uhr beginnt Lea am Donnerstag?

<details>
<summary>Answers</summary>

1. Von **acht** bis **dreizehn Uhr**. (8:00–13:00)
2. Von **halb elf** bis **halb sieben** (10:30–18:30) — 8 hours / acht Stunden.
3. **Nein**, am Mittwoch hat sie keine Uni.
4. Sie beginnt um **acht Uhr**.

</details>
