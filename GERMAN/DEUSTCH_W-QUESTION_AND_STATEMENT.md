# W-Fragen & Aussagesätze – W-Questions & Statement Word Order

> **Level:** A1 | **Netzwerk Neu A1 Reference:** Kursbuch Kapitel 1–3, Arbeitsbuch Kapitel 1–3 (p. 9, 11, 35)

---

## Overview

German sentence structure follows strict rules about **verb position**. The core rule is:

> **The verb is ALWAYS in position 2** in a statement.

For questions, the structure depends on whether it's a **yes/no question** or a **W-question** (who, what, where, when, why, how).

---

## 1. W-Fragen — Question Words

All question words begin with **W** (like English *wh-* words):

| German | English | Used to ask about |
|--------|---------|------------------|
| **wer** | who | person (subject — Nominativ) |
| **wen** | whom | person (direct object — Akkusativ) |
| **wem** | to whom | person (indirect object — Dativ) |
| **was** | what | thing / action |
| **wo** | where | location (static) |
| **wohin** | where to | direction (movement away) |
| **woher** | where from | origin |
| **wann** | when | time |
| **wie** | how | manner / description |
| **wie viel** | how much | quantity (uncountable) |
| **wie viele** | how many | quantity (countable) |
| **wie lange** | how long | duration |
| **warum** | why | reason |
| **welch-** | which | selection (takes gender endings) |

From Arbeitsbuch p. 9:
```
wer   → Wer bist du?
wo    → Wo wohnst du?
wie   → Wie heißt du?
woher → Woher kommst du?
```

---

## 2. W-Frage Word Order

```
W-WORD  +  VERB (conjugated)  +  SUBJECT  +  REST
Woher      kommst              du          ?
Wie        heißen              Sie         ?
Wann       fängt               der Kurs    an?
Wo         wohnt               er          in Berlin?
```

> The W-word takes **position 1**, the verb stays in **position 2**, and the subject shifts to **position 3**.

### Examples

| Question | Word order |
|---------|-----------|
| Wo wohnst du? | Wo (1) + wohnst (2) + du (3) |
| Woher kommst du? | Woher (1) + kommst (2) + du (3) |
| Wie heißen Sie? | Wie (1) + heißen (2) + Sie (3) |
| Was machst du? | Was (1) + machst (2) + du (3) |
| Wann hat sie Geburtstag? | Wann (1) + hat (2) + sie (3) |
| Warum lernst du Deutsch? | Warum (1) + lernst (2) + du (3) |
| Wie viel kostet das? | Wie viel (1) + kostet (2) + das (3) |

---

## 3. welch- — Which (with endings)

*Welch-* takes the same endings as the definite article:

| | Maskulinum | Femininum | Neutrum | Plural |
|--|--|--|--|--|
| Nom. | welcher | welche | welches | welche |
| Akk. | welchen | welche | welches | welche |

```
Welcher Zug fährt nach Berlin?   → Which train goes to Berlin?
Welche Jacke kaufst du?          → Which jacket are you buying?
Welches Hotel ist das?           → Which hotel is that?
```

---

## 4. Ja/Nein-Fragen — Yes/No Questions

Verb moves to **position 1** (subject goes to position 2):

```
VERB (conjugated)  +  SUBJECT  +  REST  +  ?
Kommst                du          aus Berlin?
Ist                   das         ein Hotel?
Haben                 Sie         einen Moment?
```

Answered with **ja** (yes), **nein** (no), or **doch** (yes, contradicting a negative):

```
Bist du Student?             → Ja, ich bin Student.
                             → Nein, ich bin kein Student.
Kommst du nicht?             → Doch, ich komme!  (contradicts the negative)
```

> **doch** = yes, when the question is negative. English has no direct equivalent.

---

## 5. Aussagesätze — Statement Word Order (Verb-Zweit)

The German rule: **the conjugated verb always occupies position 2**, no matter what is in position 1.

```
Position 1       Position 2   Position 3+
Ich              komme        aus Berlin.
Heute            komme        ich aus Berlin.
Aus Berlin       komme        ich.
```

> If anything other than the subject is in position 1, the subject shifts after the verb — called **Inversion**.

### Normal Order (Subject first)

```
Ich   wohne   in München.
Sie   lernt   Deutsch.
Wir   fahren  am Montag nach Wien.
```

### Inverted Order (Time / Place / Adverb first)

```
Am Montag   fährt   er    nach Wien.
Morgen      komme   ich   nicht.
In Berlin   wohnt   sie   nicht mehr.
Jetzt       lerne   ich   Deutsch.
```

> The verb **never** moves from position 2 — only the subject and other elements rearrange.

---

## 6. Full Question-Type Comparison

| Type | Structure | Example |
|------|-----------|---------|
| **Statement** | Subject + Verb + Rest | Ich komme aus Berlin. |
| **Yes/No Question** | Verb + Subject + Rest? | Kommst du aus Berlin? |
| **W-Question** | W-word + Verb + Subject + Rest? | Woher kommst du? |
| **Inverted Statement** | Time/Place + Verb + Subject + Rest | Heute komme ich. |

## 7. Common W-Question Mini-Dialogues

From Arbeitsbuch p. 9 & 11:

```

→ Ich bin Sky / Matti.

Woher kommst du?
→ Ich komme aus Jakarta / Deutschland.

Wo wohnst du?
→ Ich wohne in Berlin / München.

Wie heißt du?
→ Ich heiße Thomas.

Was machst du?
→ Ich lerne Deutsch. / Ich bin Student.

Wie ist deine Handynummer?
→ Meine Nummer ist 0878...

Wie viel kostet das?
→ Das kostet zehn Euro.

Wann hat der Kurs Geburtstag?

```

## 8. negation in questions and answers

| Positive question | Negative question | Answer with doch |
|--|--|--|
| Kommst du? | Kommst du nicht? | Doch, ich komme! |
| Hast du Zeit? | Hast du keine Zeit? | Doch, ich habe Zeit! |

## Common Pitfalls for English Speakers

1. **Subject-verb inversion is mandatory** when anything precedes the subject:
   ```
   Morgen ich komme.     ✗
   Morgen komme ich.     ✓
   ```

2. **Do-support doesn't exist in German** — no extra "do/does" in questions:
   ```
   Kommst du?    ✓   (not: Tust du kommen?)
   Wo wohnst du? ✓   (not: Wo wohnst du wohnen?)
   ```

3. **wo vs. wohin vs. woher:**
   - *wo* = where (static location): Wo wohnst du?
   - *wohin* = where to (direction): Wohin fährst du?
   - *woher* = where from (origin): Woher kommst du?

4. **wer = who (subject), wen = whom (object):**
   ```
   Wer kommt?         → Who is coming?     (subject)
   Wen liebst du?     → Whom do you love?  (object)
   ```

5. **doch** contradicts a negative question — crucial for natural German:
   ```
   Kommst du nicht? – Doch!   ✓   (Yes I am! — despite the negative question)
   Kommst du nicht? – Ja!     ✗   (sounds ambiguous)
   ```

6. **wie viel vs. wie viele:** *wie viel* = how much (no plural); *wie viele* = how many (with plural noun):
   ```
   Wie viel kostet das?       → How much does it cost?
   Wie viele Studenten sind da? → How many students are there?
   ```

### Exercise 1 — Match the W-word

Match the question word to what it asks about:

| W-word | Asks about |
|--------|-----------|
| 1. wann | A. person (subject) |
| 2. woher | B. time |
| 3. wer | C. origin |
| 4. wie | D. manner / description |
| 5. was | E. thing |
| 6. wo | F. location |

<summary>Answers</summary>

1 → B | 2 → C | 3 → A | 4 → D | 5 → E | 6 → F

### Exercise 2 — Form W-questions

Write the question that matches the underlined answer:

1. Ich heiße **Thomas**. → Wie _____?
2. Ich komme aus **Spanien**. → Woher _____?
3. Ich wohne **in München**. → Wo _____?
4. Der Kurs beginnt **um neun Uhr**. → Wann _____?
5. Das kostet **zwanzig Euro**. → Wie viel _____?
6. Ich lerne Deutsch, **weil es interessant ist**. → Warum _____?

<details>
<summary>Answers</summary>

1. Wie **heißt du / heißen Sie**?
2. Woher **kommst du / kommen Sie**?
3. Wo **wohnst du / wohnen Sie**?
4. Wann **beginnt der Kurs**?
5. Wie viel **kostet das**?
6. Warum **lernst du Deutsch**?

</details>

### Exercise 3 — Statement or inversion?

Rewrite with the time expression first:

1. Ich fahre morgen nach Berlin.
   → Morgen _____
2. Er lernt jeden Tag Deutsch.
   → Jeden Tag _____
3. Wir gehen heute Abend ins Kino.
   → Heute Abend _____

<details>
<summary>Answers</summary>

1. Morgen **fahre ich** nach Berlin.

3. Heute Abend **gehen wir** ins Kino.

</details>

### Exercise 4 — Yes/No questions + answers with ja/nein/doch

1. _____ du Student? (sind / Nominativ — formal)
   - Ja: _____
   - Nein: _____
2. _____ du kein Deutsch? (sprichst)
   - Doch: _____
3. _____ das ein Hotel? (ist)
   - Nein: _____

<details>
<summary>Answers</summary>

1. **Sind** Sie Student? / Ja, **ich bin** Student. / Nein, ich **bin kein** Student.
2. **Sprichst** du kein Deutsch? / Doch, ich **spreche** (ein bisschen) Deutsch!
3. **Ist** das ein Hotel? / Nein, das **ist kein** Hotel.

</details>
